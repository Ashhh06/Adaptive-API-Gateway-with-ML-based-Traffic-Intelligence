# Dataset (not in git)

CSV and zip files here are ignored so the repo stays small and clones stay fast.

To train: get the CIC-IDS-2017 **MachineLearningCSV** (or compatible) archives, extract under this folder so `train_cic_gateway_features.py` can find `**/*.csv`. Same idea if you use other zips your script expects.

`.md5` files can stay local for checksum checks; they are small if you choose to commit them.
